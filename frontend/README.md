# react-chat

## Установка и использование

- `yarn install` - установка зависимостей
- `yarn dev` - запуск дев сервера с хотрелоадом
- `yarn ui` - запуск storybook
- `yarn ui:build` - сборка storybook
- `yarn build` - сборка бандла
